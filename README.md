# Crypto-wallet info:

This project is an example of integrating a crypto wallet into a react application. You can connect a metamask wallet and register, then see yourself in the list of users

# Stack:

- TypeScript
- React
- RTK
- RTK Query
- useDapp
- React hook form
- styled-components
- MUI
